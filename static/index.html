АС-УчЭнерг

Автоматизированная система учёта и анализа электроэнергии

О проекте

Проект «АС-УчЭнерг» разработан студентами Уфимского университета науки и технологий в рамках проекта по дисциплине «Администрирование информационных систем». Цель — создание клиент-серверной системы с полным жизненным циклом, охватывающей этапы от анализа требований до реализации и оценки качества.

Актуальность

  • Энергоэффективность: контроль потребления, выявление аномалий и несанкционированного использования.
  • Экономия ресурсов: оптимизация затрат на электроэнергию и использование выгодных тарифов.
  • Юридическая прозрачность: соответствие требованиям законодательства и упрощение отчётности.
  • Управление: оперативный доступ к данным и аналитике для принятия решений.
  • Удобство для пользователей: личный кабинет, прозрачные расчёты и история потребления.

Функциональные возможности

По ролям:

  • Арендатор/жилец:
    • Просмотр истории показаний и начислений
    • Просмотр текущего тарифа
  • Бухгалтер/управляющий:
    • Просмотр всех показаний
    • Изменение тарифов
    • Формирование отчётов
  • Администратор:
    • Полный CRUD-доступ ко всем сущностям: пользователи, здания, счётчики и т.д.

Технологический стек

  • Бэкенд: Python, Flask, SQLAlchemy
  • База данных: MySQL
  • Фронтенд: HTML5, CSS3, JavaScript (GUI-клиент на PyQt6)
  • Инфраструктура: Docker, Docker Compose

Архитектура и моделирование

  • Моделирование бизнес-процессов: IDEF0, IDEF3
  • Проектирование архитектуры: UML (диаграммы прецедентов, классов, последовательности, состояний)
  • Клиент-серверная архитектура с RESTful API
  • Ролевая система доступа и аутентификация

Результаты тестирования

  • Функциональность: реализовано 100% требований из ТЗ
  • Надёжность: коэффициент готовности — 99.84%
  • Нагрузочное тестирование (Locust): стабильная работа при 20 параллельных пользователях
  • Стресс-тест: точка отказа — свыше 200 пользователей
